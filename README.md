# OVERVIEW 

* CSS・SCSS勉強用
* コンポーネント・UIを作成


# CONFIGURE

```
app
├ dist
│   └ app.js ... ビルドされたJS
│
├ src
│   ├ index.js ... ビルドされたJS
│   ├ index.scss ... 汎用的なスタイルシート
│   ├ index.css ... ビルドされたCSS
│   └ XXX_Page ... 各種ページのJS(JSX)、SCSSが配置されている
│
├ ui
│   └ XXXX ... ボタンやテキスト等の汎用的なUIコンポーネント
│   
└ index.html
```

# RUN

```
npm start # ブラウザから8000ポートアクセス
```
